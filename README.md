# Network Traffic Incident Reconstruction and Alert Analysis Report

This repository contains a professional technical assessment report based on multi-session packet analysis and alert review using Wireshark and Snort.

## Project Summary
The assessment reviewed four related packet captures to reconstruct suspicious network activity, analyze protocol behavior, and correlate packet evidence with alert output.

The report focused on:
- session timeline comparison
- protocol exposure across multiple captures
- plaintext Telnet authentication review
- password file access and transfer activity
- Snort alert correlation
- reconstruction of likely suspicious activity across internal hosts

## Key Findings
- Capture 3 contained the most significant evidence and was the primary focus of the analysis.
- Telnet traffic exposed credentials in plaintext between internal systems.
- The captures showed suspicious host interaction and password file handling behavior.
- Snort alerts supported the presence of reconnaissance and suspicious network activity.
- The report documented observed evidence conservatively and avoided overstating unsupported attack conclusions.
